BadgeUp Ionic Client Demo

BadgeUp Ionic Client Demo App

This application demonstrates the out-of-the box functionality of the BadgeUp Ionic client, @badgeup/badgeup-ionic-client

Interaction

Open the achievement overview below by clicking the button below.

In an actual app, the overview might be a tab or dedicated page.

This button sends an event, achievement-please that may earn an achievement. This represents a user interaction event.

When an achievement is earned, a popup is presented by the ionic client.
